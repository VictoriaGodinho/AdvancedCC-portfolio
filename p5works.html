p5.js Works

These are projects created with p5.js, a JavaScript library that makes creative coding accessible for artists, designers, and beginners. It allows you to easily combine code with visual art, animation, and interaction directly in the browser. My works here explore generative visuals, interactive typography, and playful experiments where you can interact with each sketch by moving the mouse, clicking, dragging, holding, or pressing keys.
